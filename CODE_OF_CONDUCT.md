# Code of Conduct

We follow the **Contributor Covenant v2.1** as the baseline standard of
behavior for contributors, reviewers, and issue commenters on this
project.

## TL;DR

- Treat every contributor with respect.
- Criticize code, not people.
- Assume good faith; ask for clarification before escalating.
- Don't post content that targets, harasses, or mocks individuals based
  on who they are.
- The maintainer may remove comments, close issues, or block users that
  violate these rules.

## Full text

The full Contributor Covenant text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
in Portuguese, English, and 40+ other languages. That document, verbatim,
is the authoritative rule set for this repository.

## Reporting

If you see behavior that violates this code of conduct, open a private
issue by emailing the maintainer (address on the GitHub profile) with
subject `[CoC]`. Reports are handled confidentially and in good faith.
Retaliation against reporters is itself a violation.

## Scope

This code applies to every space the project uses: GitHub issues, pull
requests, discussions, commit messages, the code itself, and any
off-platform spaces where a contributor represents the project.
